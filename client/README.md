# OnFinance AI - Client

This directory contains the React + Vite frontend for the UML AI Generator.

For full project documentation, setup instructions, and feature details, please refer to the [Main README](../README.md) in the project root.

## Quick Start (from this directory)

1. Install dependencies:
   ```bash
   npm install
   ```
2. Set up your environment variables in `.env` (see `.env.example`).
3. Run the development server:
   ```bash
   npm run dev
   ```
